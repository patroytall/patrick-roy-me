# Ubuntu

## Install

Create USB key: https://tutorials.ubuntu.com/tutorial/tutorial-create-a-usb-stick-on-ubuntu#0

Change boot order and reboot with USB key.

In Firefox, "download chrome". Follow procedure to install Chrome.

Remove Firefox: https://askubuntu.com/questions/16758/removing-firefox-in-ubuntu-with-all-add-ons-like-it-never-existed

## Optional - Infinite sudo

Run sudo visudo and add this line:

Defaults timestamp_timeout=-1

## Optional - Hide Dock

Settings -> Dock → Auto-hide the Dock -> On

## Optional - Hide Top Bar

`sudo apt install chrome-gnome-shell`

Click on GNOME Shell Extension (foot) icon in Chrome beside URL field.

Install "Hide Top Bar". Click extension Configure icon. Enable "Show panel when mouse approaches edge of screen".

## Optional - Using multiple workspaces

Ctrl + Alt + [Down and Up] to move between workspaces

To move a window from space to space: Ctrl + Alt + Shift + [Arrow Keys]
